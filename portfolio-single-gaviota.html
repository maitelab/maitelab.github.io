Gaviota

G

Web extractora de noticias.

Descripción:

Es una página web que permite a personas con/sin discapacidad visual extraer el texto de una página web que contenga una noticia, facilitando su lectura sin anuncios, imágenes u otros elementos que obstruyen el uso de un lector de pantalla o su fácil lectura. Además genera automáticamente un resumen de la noticia para que pueda ser revisada rápidamente para determinar si vale la pena leer el artículo entero.

Función:
