Maitenotas

M

Software de diario electrónico seguro.

Descripción:

Aplicación de escritorio para guardar notas de manera ordenada según un esquema de árbol, puede ser utilizada como journal o diario electrónico. La información se guarda encriptada con contraseña. Fue diseñada para que sea compatible con lectores de pantalla. Para probarla descarga el archivo maitenotas_renombrar_a_punto_exe.zip desde esta carpeta, luego descomprimir utilizando el password "manzana" y el renombrar a .exe. La palabra "maitenotas" nace de la unión de Maiten y "notas" (notas guardadas en un árbol).

Tipo
App
Función:

Diario electrónico
