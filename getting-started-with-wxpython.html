Getting started with wxPython

By Taksan, mar 01 enero 2019, modified mar 01 enero 2019, in category Python

GUI, python

There are already a lot of tutorials on internet about how to make apps with wxPython, but they don't tell you how to get started from zero, so we will cover that here.

How to instal wxPython in windows

Just do:

pip install wxpython

How to instal wxPython in linux

We will be installing GTK first (needed by wxPython in linux I assume)

sudo apt-get update
sudo apt-get install build-essential gnome-devel
sudo apt install python-gobject python3-dev

sudo apt-get install libwebkitgtk-dev libjpeg-dev libtiff-dev libgtk2.0-dev libsdl1.2-dev freeglut3 freeglut3-dev libnotify-dev

Next, libgstreamer... if you are in ubuntu, do this:

sudo apt-get install libgstreamerd-3-dev

But if you are in ubuntu, do this:

sudo apt-get install libgstreamer*

Next:

sudo apt-get install libwebkitgtk*

Finally (prepare some coffee because this last step will take some time to complete) and do:

pip install python-config
pip install wxpython

Hello world in wxPython

import wx
app = wx.App(False)  # Create a new app, don't redirect stdout/stderr to a window.
frame = wx.Frame(None, wx.ID_ANY, "Hello World!") # A Frame is a top-level window.
frame.Show(True)     # Show the frame.
app.MainLoop()

From here, the best place to go for is the wxPython github, look inside the examples and demo folders, you will get plenty of easy to understand code that you can learn from.
