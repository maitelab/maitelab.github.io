Getting started with QT and python

By Taksan, jue 03 enero 2019, modified jue 03 enero 2019, in category Python

GUI, python, QT

There are already a lot of tutorials on internet about how to make apps with QT and python, but they don't tell you how to get started from zero, so we will cover that here.

We must first stablish the difference between "PyQT" and "QT for Python":

GUI library Features
PyQT
  • Mature (many years of history)
  • Lots of books and examples available
  • Not free for commercial products
  • Made by a third party company
QT For Python
  • New (First version from around 2018)
  • Little documentation available
  • Free for commercial products
  • Official version from QT

This post is about QT for python.

How to install QT for python

First download the wheel for your operating system from https://download.qt.io/official_releases/QtForPython/pyside2/

After you downloaded the latest version of the wheel for your operating system (and python version), install it with pip. Here are two examples on how to do that (one for linux, the other for windows, just execute the one you need):

pip install PySide2-5.11.2-5.11.2-cp35.cp36.cp37-abi3-manylinux1_x86_64.whl

pip install PySide2-5.11.2-5.11.2-cp35.cp36.cp37-none-win_amd64.whl

If you want to run the OpenGL examples, also install:

pip install PyOpenGL PyOpenGL_accelerate

Hello world in QT for Python

This example was copied from: https://doc.qt.io/qtforpython/tutorials/basictutorial/clickablebutton.html

#!/usr/bin/python

import sys
from PySide2.QtWidgets import QApplication, QPushButton
from PySide2.QtCore import Slot

@Slot()
def say_hello():
print("Button clicked, Hello!")

# Create the Qt Application
app = QApplication(sys.argv)
# Create a button, connect it and show it
button = QPushButton("Click me")
button.clicked.connect(say_hello)
button.show()
# Run the main Qt loop
app.exec_()

To learn more, visit the examples that the wheel installed in your computer, they are located in the "site-packages" folder of your python environment ( usually something like: python3.6/site-packages/PySide2/examples ).
