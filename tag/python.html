Generar un ejecutable de CEF Python

lun 01 julio 2019 | tags: python

1: Instalar python+cx_freeze

En Windows de preferencia usar usar python 3.6.5-amd64.exe pues esa versión funciona ok con cx_freeze. Instalar también pywin (pywin32-224.win-amd64-py3.6.exe) que se puede descargar de aquí

Luego instalar cx_Freeze con pip

python -m pip install cx_Freeze --upgrade

2: Crear un setup.py (Linux)

Suponiendo que el programa principal es "cef_hello_world.py", entonces su configuración para cx_freeze (en linux) será:

from cx_Freeze import setup, Executable

setup(name = "myapp" ,
      version = "0.1" ,
      description = "my super app" ,
      requires = ["requests"],
      options = {
          "build_exe": {
          'packages': ["os","sys","ctypes", "json","urllib", "multiprocessing"],
          'includes': ["idna.idnadata"],
          'include_msvcr': True,
      }},
      executables = [Executable("cef_hello_world.py")])

2: Crear un setup.py (Windows)

Suponiendo que el programa principal es "cef_hello_world.py".

Hay que usar el archivo que está recomendado en su web

Crea la carpeta otherResourceFolder dentro del proyecto.

Configuración setup.py:

from distutils.sysconfig import get_python_lib
from os.path import join
import os
from glob import glob
from cx_Freeze import setup, Executable

#indica la ubicacion de las TCL
os.environ['TCL_LIBRARY'] = r'C:\Python36\tcl\tcl8.6'
os.environ['TK_LIBRARY'] = r'C:\Python36\tcl\tk8.6'

# Basically just copy all of the CEF distribution into the installer
# I have only tested this on windows
cefPath = join(get_python_lib(), "cefpython3")
CEF_INCLUDES = glob(join(cefPath, "*"))
CEF_INCLUDES.remove(join(cefPath, "examples"))

setup(
    name = "MyAppName",
    version = "1.0.0",
    options = {
        "build_exe": {
        'packages': ["os","sys","ctypes","win32con", "json"],
        'include_files': CEF_INCLUDES + ['otherResourceFolder'],
        'include_msvcr': True,
    }},
    executables = [Executable("cef_hello_world.py", base="Win32GUI")]
)

3: Lanzar la generación del empaquetado

python setup.py build

Si todo ejecutó correctamente... en la carpeta build estará el ejecutable (.exe en el caso de Windows) con el programa que se puede distribuir a otra computadora.
